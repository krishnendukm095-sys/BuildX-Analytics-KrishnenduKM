# BuildX Analytics Project – Zomato Restaurant Analysis

## Dataset

Zomato Restaurants India Dataset

## Why I Selected This Dataset

I selected the Zomato Restaurants dataset because it provides real-world information on restaurant ratings, costs, online ordering, and customer preferences, making it ideal for data analysis.



### 1. Which restaurant type has the highest number of restaurants?
**Answer:** Quick Bites has the highest number of restaurants (**2,840**).

### 2. What percentage of restaurants provide online ordering and table booking?
**Answer:** **52.46%** provide online ordering, while **10.47%** offer table booking.

### 3. What is the relationship between restaurant ratings and average cost?
**Answer:** Higher-priced restaurants do not always have higher ratings. Good ratings are found across different price ranges.

## Tools Used

- Python (Pandas)
- SQLite
- Power BI
- GitHub

## Key Insight

Quick Bites is the most common restaurant type. More than half (**52.46%**) of restaurants support online ordering, but only **10.47%** offer table booking. The average restaurant rating is **3.51/5**, and the average cost for two people is **₹539**.
