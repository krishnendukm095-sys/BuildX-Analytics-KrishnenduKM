# BuildX Analytics Project – Zomato Restaurant Analysis

## Dataset

Zomato Restaurants India Dataset

## Why I Selected This Dataset

I selected the Zomato restaurant dataset because it provides valuable information about restaurant ratings, costs, online ordering, and customer preferences. It is a realistic business dataset that allows meaningful analysis using Python, SQL, and Power BI.

## Business Questions

1. Which restaurant type has the highest number of restaurants?
2. What percentage of restaurants provide online ordering and table booking?
3. What is the relationship between restaurant ratings and average cost?

## Tools Used

- Python (Pandas)
- SQLite
- Power BI
- GitHub

## Key Insight

Quick Bites is the most common restaurant type with 2,840 restaurants. Around 52.46% of restaurants provide online ordering, while only 10.47% offer table booking. The average restaurant rating is 3.51/5, and the average cost for two people is ₹539.
